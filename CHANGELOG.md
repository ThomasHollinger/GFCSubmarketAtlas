# Changelog

## v2.8.2 - Builder Starts Detail

- Builder subdivision pins no longer zoom the map on single click.
- Builder subdivision pins now zoom only on double click, matching school pin behavior.
- Added a new Builder Subdivisions sidebar dropdown: Starts by Builder.
- Starts by Builder shows each builder's annual starts and share of annual starts inside the selected boundary.
- Preserves builder colors, builder initials, verified school ratings, ACS 2020-2024 demographics, healthcare, and retail filters.
