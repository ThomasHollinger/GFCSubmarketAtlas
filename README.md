# Gulf Coast Submarket Atlas v3.0.0

This release rebuilds the atlas into a modular application structure.

## Key changes
- Modular JavaScript files for map, schools, search, sidebar, themes, and data loading.
- Market Panel layout based on the requested sidebar order.
- School Rating calculations run in the background and do not require school pins to be visible.
- Schools on = School Rating heat map + school pins.
- Schools off = Hub View.
- Missing ratings export tool retained.

## Deploy
Upload all files/folders in this ZIP to the repository root.
